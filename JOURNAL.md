# June 18

An ETC Ion Xe costs over $10,000, I want to build my own for cheaper. Fortunately, with a student ETC Nomad key and USB to DMX converter, I can have all the features of the Ion Xe for just $350. The catch? I still don't have any of the creature comforts of a real console, namely the custom keys, faders, and encoders that make programming lights a breeze. I'm trying to build a custom control surface that adds this functionality and can be used with any ETC Eos software (even an existing console! Let someone else learn with you, or make your Ion more like an Ion Xe!).

Today, I tried to see how feasible this really is, worked on a BOM, and started to make plans for the PCB. I'm going to try and break it up into multiple PCBs for cost, and handwire individual components not part of a matrix. I'm thinking of Gateron black switches, as it's similar to what's used on a real console, and is fairly cost effective. Next up, I want to try and select some more parts, get a cost estimate, and then I can start designing!

# June 19

Today I kept working on the BOM to try and get a more concrete idea on what the prices of parts will be, and how everything is going to fit together. I'm still a bit stuck on how I'm going to end up connecting all these PCBs together, but that will be a problem for the future. I also still need to figure out this whole situation with the softkeys. I think I might just be able to have buttons? We'll see. I'm still dissapointed that I wasn't able to find a good smooth encoder for cheap, I could only find one on mouser that's like $2 each, and I need 6. Shipping also adds a bunch of extra cost so maybe I'll buy some other parts there too to save on shipping. Right now the BOM is sitting around 100 bucks, but the PCB cost is probably inaccurate, and I also need to add in softkeys and filament costs. Hopefully I can get everything under like ~150 so I won't need to spend too many hours on this project.
